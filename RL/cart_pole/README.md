# CartPole — REINFORCE с baseline (policy gradient)

REINFORCE (Monte-Carlo policy gradient) на PyTorch для среды `CartPole-v1`,
с опциональным **baseline** (value-critic) для снижения дисперсии градиента.
Главный эксперимент — сравнение обучения **с baseline и без**.

## Архитектура

- **Actor** (`PolicyNet`) — MLP `state(4) → hidden(128) → logits(2)`, задаёт политику `π(a|s)`
- **Critic** (`ValueNet`) — MLP `state(4) → hidden → V(s)`, оценка ценности состояния (создаётся только при `use_baseline=True`)

## Алгоритм

- Собираем полный эпизод (Monte-Carlo), считаем дисконтированные returns `G_t`
- Обновление политики: максимизируем `E[ log π(a|s) · advantage ]`
- Advantage:
  - **с baseline**: `A_t = G_t − V(s_t)`, критик учится приближать `G_t` через MSE
  - **без baseline**: `A_t = G_t` (чистый REINFORCE)

## Эксперимент: baseline vs no baseline

Центральный вывод — роль baseline как снижателя дисперсии policy gradient:

Обе конфигурации — бюджет 1000 эпизодов:

| `use_baseline` | Результат |
|----------------|-----------|
| **True** (REINFORCE + baseline) | стабильно доходит **до 500** (идеальный максимум `CartPole-v1`) и решает задачу — ранняя остановка срабатывает до конца бюджета |
| **False** (vanilla REINFORCE) | за **1000 эпизодов** упирается в **~300** и задачу так и не решает — обучение шумное и нестабильное |

**Почему:** без baseline градиент имеет высокую дисперсию — отдельные «удачные»
эпизоды тянут веса хаотично. Вычитание `V(s)` центрирует advantage вокруг нуля:
действия лучше среднего усиливаются, хуже — подавляются. Дисперсия падает, и
обучение доходит до потолка среды.

> `CartPole-v1` обрывает эпизод на 500 шагах, а reward = число шагов, что шест
> продержался. Поэтому reward 500 = идеальная политика.

## Гиперпараметры (по умолчанию)

| Параметр | Значение | Смысл |
|----------|----------|-------|
| `gamma` | 0.99 | дисконт будущих наград |
| `lr_actor` | 1e-3 | learning rate политики |
| `lr_critic` | 1e-3 | learning rate критика |
| `hidden` | 128 | ширина скрытого слоя |
| порог «решено» | mean(100) ≥ 475 | ранняя остановка |

## Запуск

```bash
python RL/cart_pole/cart_pole.py
```

Зависимости: `gymnasium[classic-control]`, `torch`, `numpy`, `matplotlib`.
Методы агента: `train()`, `evaluate()`, `plot_progress()`, а также `watch()` —
визуализация политики в отдельном окне.
