# FrozenLake — Q-learning агент

Табличный Q-learning агент для среды `FrozenLake-v1` (Gymnasium), реализованный
с нуля на NumPy. Агент учится дойти от старта до цели по замёрзшему озеру, не
проваливаясь в проруби.

## Подход

- **Q-таблица** `(n_states × n_actions)` — оценка ценности каждого действия в каждом состоянии
- **ε-greedy** исследование с экспоненциальным затуханием `epsilon` (1.0 → 0.02)
- **TD-обновление**: `Q[s,a] += α · (r + γ · max Q[s',·] − Q[s,a])`
- Честная оценка политики (`evaluate`) — жадный `argmax` по Q без исследования (RL-аналог `model.eval()`)

## Режимы

- **Детерминированный** (`is_slippery=False`) — действие всегда выполняется, задача простая
- **Стохастический** (`is_slippery=True`) — агент случайно «проскальзывает» вбок; часть падений в проруби неизбежна, поэтому success rate физически не может достичь 1.0

**Результат в стохастическом режиме: success rate (eval) = 0.72** — сильный результат для скользкого поля 4×4.

## Гиперпараметры (по умолчанию)

| Параметр | Значение | Смысл |
|----------|----------|-------|
| `alpha` | 0.1 | скорость обучения |
| `gamma` | 0.99 | дисконт будущих наград |
| `epsilon` | 1.0 → 0.02 | доля случайных действий |
| `episodes` | 50000 | эпизодов обучения (скользкий режим) |

## Запуск

```bash
python RL/frozen_lake/frozen_lake.py
```

Зависимости: `gymnasium`, `numpy`, `matplotlib`. Скрипт обучает агента, строит
графики обучения, печатает выученную политику стрелками и Q-таблицу.
