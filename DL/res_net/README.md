# ResNet-18 — классификация CIFAR-10

Реализация ResNet-18 **с нуля** на PyTorch для классификации изображений CIFAR-10.
Проект учебный: цель — понять архитектуру остаточных сетей (residual connections,
downsampling, global average pooling) и собрать полный пайплайн обучения с
аугментациями, планировщиком learning rate и ранней остановкой.

## Архитектура

Классический ResNet-18: 4 стадии по 2 базовых блока, каналы 64 → 128 → 256 → 512,
downsampling через `stride=2` в первом блоке каждой стадии, финальный
global average pooling + полносвязный слой на 10 классов.

Базовый блок (`BasicBlock`):

```
x ──► conv3x3 ─► BN ─► ReLU ─► conv3x3 ─► BN ─► (+) ─► ReLU ─►
  └────────────────── shortcut ──────────────────┘
```

Shortcut — identity, если размерность и число каналов не меняются, иначе проекция
через `1x1 conv` + BN. Остаточная связь (`out + shortcut(x)`) позволяет градиенту
течь в обход, что делает возможным обучение глубоких сетей без затухания градиента.

## Возможности

- ResNet-18 полностью с нуля (без `torchvision.models`)
- Аугментации: `RandomCrop(32, padding=4)`, `RandomHorizontalFlip`, нормализация
  статистиками CIFAR-10
- Валидация без аугментации (отдельный transform) — чтобы оценка не была зашумлена
- SGD + momentum + `CosineAnnealingLR` — классические инструменты для ResNet/CIFAR
- Ранняя остановка с восстановлением лучших весов
- Трекинг loss и accuracy, построение графиков
- Автоматическая загрузка датасета через torchvision


## Основные классы:

- `BasicBlock` — остаточный блок из двух свёрток + shortcut
- `ResNet18` — сборка блоков в полную сеть
- `ResNetFinal` — обёртка: загрузка данных, аугментации, обучение, оценка, графики

## Запуск

```python
m = ResNetFinal()
m.train_model(epochs=60)
m.plot_losses()
print("test accuracy:", m.test_accuracy())
```

## Данные

CIFAR-10: 60 000 цветных изображений 32×32 в 10 классах (самолёт, автомобиль, птица,
кот, олень, собака, лягушка, лошадь, корабль, грузовик), 50 000 train + 10 000 test.
Из обучающей выборки 10% выделяется под валидацию.

Источник: `torchvision.datasets.CIFAR10` (скачивается автоматически) либо
официальная страница `cs.toronto.edu/~kriz/cifar.html`.

## Результаты

С настройками по умолчанию (SGD momentum, cosine LR, ~60 эпох) точность на
тестовой выборке — 0.9399. Это типичный результат для ResNet-18 с нуля на
CIFAR-10 с базовыми аугментациями.

## Ключевые идеи

- **Residual connections** — `out + shortcut(x)` даёт градиенту обходной путь,
  ключ к обучению глубоких сетей
- **BatchNorm после каждой свёртки** — стабилизирует обучение, `bias=False` в
  свёртках (BN всё равно вычитает среднее)
- **Global average pooling** вместо flatten + большого FC — резко меньше параметров,
  действует как регуляризация
- **Cosine LR schedule** — плавное снижение learning rate к концу обучения улучшает
  сходимость по сравнению с фиксированным lr
