# Bigram Language Model — Statistical & Neural Network

Два подхода к предсказанию следующего символа в имени, реализованные с нуля, по мотивам [Karpathy's makemore](https://github.com/karpathy/makemore). Задача — научиться генерировать правдоподобные имена на уровне символов.

## Что реализовано

### 1. Статистическая модель (`NameGenModel`)
Классический bigram-подход: подсчёт частот переходов между символами и нормализация в вероятности.
Loss = 2.4734

- Построение словаря символов (`stoi`/`itos`) из датасета имён
- Матрица частот биграмм `N[ch1, ch2]` — сколько раз символ `ch2` встречался после `ch1`
- Генерация новых имён через сэмплирование (`torch.multinomial`)
- Оценка качества через **negative log-likelihood (NLL)** — среднее значение `-log(P)` по всем биграммам датасета

### 2. Нейросетевая модель (`BigramNNModel`)
Та же задача, но веса не считаются напрямую, а обучаются градиентным спуском. Loss = 2.4966 стремится к потерям статической модели

- Один линейный слой (`nn.Linear`, без bias) — обучаемый аналог матрицы `P`
- One-hot encoding входного символа
- `F.cross_entropy` (softmax + NLL) в качестве функции потерь + L2-регуляризация как аналог сглаживания
- Ручной цикл обучения (forward → loss → backward → update весов) — реализован явно, без `torch.optim`, чтобы видеть механику
- Та же метрика NLL для сравнения со статистической моделью

## Ключевой вывод

Обе модели решают математически одну и ту же задачу — максимизацию правдоподобия следующего символа — просто разными методами: статистика считает оптимальные вероятности аналитически (через частоты), нейросеть находит их итеративно (через градиентный спуск). При сопоставимой настройке (сглаживание ≈ регуляризация) **NLL обеих моделей сходится к близким значениям**, что подтверждает эквивалентность подходов.

## Пример использования
```python
# Статистическая модель
stat_model = NameGenModel("names_table.jsonl", seed=30)
print(stat_model.generate(10))
print("NLL:", stat_model.nll_loss())

# Нейросетевая модель
nn_model = BigramNNModel("names_table.jsonl", seed=30)
nn_model.train_model(epochs=200, lr=10.0)
print(nn_model.generate(10))
print("NLL:", nn_model.nll_loss())
```
